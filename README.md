## Getting Started

To get started with Scribble Blinks, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/kitsunekode/scribble-blinks.git
    ```
2. Navigate to the project directory:
    ```sh
    cd scribble-blinks
    ```
3. Install the dependencies:
    ```sh
    npm install
    ```
4. Run the project:
    ```sh
    npm run dev
    ```
